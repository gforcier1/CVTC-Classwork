Problem Statement

A program is needed to calculate the time it will take for an object to hit the ground when dropped from a given height. The program will prompt for the height in feet, and then compute and display the number of seconds it takes for the object to hit the ground, using the basic Physics formula for "falling bodies." (Air resistance and friction are neglected; all objects fall at the same rate of acceleration.)

t = square root of (2d / g)

d is distance, t is time, g is the pull of gravity (acceleration) on the object. DO NOT use "d", "t", and "g" as variable names in your code.

Assume g = 32 feet per second per second

To do the square root of a number in JavaScript, you use Math.sqrt(); 

Math.sqrt(9); // square root of 9 is 3.

Additional Requirements

  • You must create a function that takes in the input from the user as an argument and computes the distance.
  • Your function must return the time back to the rest of your program.
  • Your function should declare and contain the variable for the acceleration of gravity.
  • Your function must not display results to the screen.
  • Your function must not prompt for input.
  • Your function must not access any variables not declared inside of the function.
  • You may choose to encapsulate your main program in a main function.
