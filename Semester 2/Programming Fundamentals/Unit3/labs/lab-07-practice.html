Problem Statement

A program is required that will receive two numbers from a user, calculate the sum, difference, product, and quotient of the numbers. The program will display the two input numbers and the calculated results with identifying text.

1. Nouns and Verbs

Identify the nouns and verbs in the problem statement. Concentrate on the important nouns and verbs that are relevant to the problem.

PROBLEM STATEMENT:

Nouns: two numbers, sum, difference, product, quotient

Verbs: receive, calculate, display

2. Defining Diagram

Convert the nouns and verbs into variable names and processes and create a defining diagram. Follow the course naming standards for inputs and outputs.

INPUTS:
numberOne
numberTwo

PROCESSING:
prompt numberOne
prompt numberTwo
calculate sum
calculate difference
calculate Product
calculate quotient
display inputs
display calculations

OUTPUTS:
display inputs
display calculations

3. Solution Algorithm

Create a solution algorithm based on the defining diagram. (Don't forget the algorithm name and END).

SOLUTION ALGORITHM

displayInputsAndCalculations
  prompt for numberOne
  prompt for numberTwo
  sum is equal to numberOne plus numberTwo
  difference is equal to numberOne subtracted by numberTwo
  product is equal to numberOne multiplied by numberTwo
  quotient is equal to numberOne divided by numberTwo
  display numberOne plus page break
  display numberTwo plus page break
  display sum plus page break
  display difference plus page break
  display product plus page break
  display quotient plus page break
  
END
