Problem Statement

A program is required that prompts a user for the first name and last name of a dog owner and the name of his or her dog. The program will generate an information statement for the dog owner using the following pattern and then display the string on the page. (Check out the sample output below.)

  • Example output: Lou Smith's dog is named "Fido".
  • These are the parts of the string:
    • First Name
    • A space
    • Last Name
    • An apostrophe
    • The letter s
    • A space
    • This string of text: dog is named
    • A space
    • A double quote mark
    • The dog's name
    • A period
    • A double quote mark
